# AURA · AI 人格配置器

> 通过 16 个问题的人格测评，为你生成专属的 AI 助手配置。

[English](./README.md) | 中文

## 🎯 AURA 是什么？

AURA 是一个交互式 Web 应用，帮助你创建定制化的 AI 人格配置。回答 16 个精心设计的问题，涵盖你的沟通风格、工作偏好和与 AI 的理想关系，我们会为你生成一份完整的系统提示词配置。

## ✨ 核心功能

- **16 题精准测评**：多维度深度人格分析
  - 互动方式（主动/平衡/被动）
  - 沟通风格（简洁/详尽/随性）
  - 反馈方式（直言/温和/共情）
  - 任务处理（计划/敏捷/参谋）
  - 还有 12 个维度...

- **多维人格分析**：
  - 5 轴人格雷达图
  - 动画化的能力条形图（主动性、清晰度、诚实度、执行力、共情力）
  - 详细的人格特征分解

- **高质量配置导出**：
  - 专业的系统提示词，包含角色定义、沟通准则、工作方式、成长建议
  - Markdown 格式，带 YAML 前置元数据
  - 开箱即用，适配任何 AI 助手

- **精美的用户界面**：
  - 深色主题 + 渐变色彩
  - 流畅的动画和过渡效果
  - 完全响应式设计（手机/平板/桌面）
  - 完成时的彩纸庆祝动画

- **多语言支持**：中文 & English

## 🚀 快速开始

### 环境要求
- Node.js 20+
- npm 或 yarn

### 安装

```bash
git clone https://github.com/DW-WJ/aura-ai.git
cd aura-ai
npm install
```

### 本地开发

```bash
npm run dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000)

### 生产构建

```bash
npm run build
npm start
```

## 🎨 技术栈

- **框架**：Next.js 16（React 19）
- **语言**：TypeScript
- **样式**：Tailwind CSS
- **动画**：CSS 动画 + Canvas（雷达图）
- **部署**：Vercel 就绪

## 🧠 工作原理

1. **测评**：用户回答 16 个跨越 8 个人格维度的问题
2. **分析**：系统计算 5 个核心指标的人格分数（0-100）
3. **生成**：AI 人格矩阵生成：
   - 独特的 AI 名字（从精选名字库中随机）
   - 人格类型标签
   - 详细的角色描述
   - 沟通准则
   - 工作方式指导
   - 成长支持策略
4. **导出**：用户可复制或下载配置为 `.md` 文件

## 📊 人格维度详解

| 维度 | 量表 | 说明 |
|------|------|------|
| **主动性** | 主动 ↔ 被动 | AI 是否应该主动提供信息 |
| **清晰度** | 简洁 ↔ 详尽 | 沟通风格偏好 |
| **诚实度** | 直言 ↔ 温和 | 反馈交付方式 |
| **执行力** | 敏捷 ↔ 计划 | 任务处理方法 |
| **共情力** | 专业 ↔ 温暖 | 情感语调 |

## 📝 使用示例

1. 打开应用，点击"开始配置"
2. 逐题回答 16 个问题（或点击"随机"快速跳过）
3. 等待 AI 生成你的专属配置
4. 查看详细分析、能力雷达图、系统提示词
5. 复制配置或下载为 Markdown 文件

## 🎁 配置文件包含

生成的 `.md` 文件包含：

- **角色定义**：你的 AI 是什么
- **服务领域**：专注的工作类型
- **沟通准则**：如何与你交流
- **工作方式**：如何处理任务
- **成长支持**：如何帮助你成长
- **记忆机制**：如何记住你
- **边界意识**：如何处理敏感话题
- **能力画像**：5 维能力评分表
